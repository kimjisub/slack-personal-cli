# Launch copy

Ready-to-post drafts for sharing `slack-personal-cli`, one file per channel.
Tone is tuned per community. All claims are kept honest (macOS-only,
session-based, not official OAuth) on purpose — overstating gets punished,
especially on HN.

| File | Channel | Notes |
|------|---------|-------|
| [`show-hn.md`](show-hn.md) | Hacker News (Show HN) | Title + first comment. Humble, technical, limitations up front. |
| [`reddit-commandline.md`](reddit-commandline.md) | r/commandline | Casual, demo-GIF-friendly, self-promo disclosed. |
| [`x-thread.md`](x-thread.md) | X / Twitter | 6-tweet thread, strong hook, attach the demo GIF. |
| [`awesome-list.md`](awesome-list.md) | awesome-* lists | One-line entry + PR blurb. |

The demo GIF lives at [`../../assets/demo/slk-demo.gif`](../../assets/demo/slk-demo.gif).
Attach it to the Reddit post and tweet 1; on HN it renders from the README.

Repo URL used throughout: <https://github.com/kimjisub/slack-personal-cli>
